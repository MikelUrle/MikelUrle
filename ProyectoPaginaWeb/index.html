Para la creacion de esta Web, se han utilizado las modernas herramientas de:

HTML5

Es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 específica dos variantes de sintaxis para HTML: una «clásica», HTML, y una variante XHTML conocida como sintaxis XHTML5 que deberá servirse con sintaxis XML

CSS 3

Es una tecnología que ha tenido una evolución en el tiempo, que actualmente se encuentra en su versión 3, como su propio nombre indica. Sus siglas corresponden a “Cascading Style Sheets”, que tiene el siguiente significado:

  • Cascading, que significa que los estilos que aplicamos a los elementos de una página web se propagan a los elementos que contiene, se propagan en cascada.
  • Style, porque mediante CSS lo que hacemos es aplicar estilos visuales a los distintos elementos de nuestra página web.
  • Sheets, que significa hojas, porque los estilos de una página web se añaden en ficheros aparte, en ficheros con la extensión .css de manera general.
